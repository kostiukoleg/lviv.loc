== Description ==

ACF for WooCommerce is a freemium WordPress plugin that combine the power of WooCommerce and Advanced Custom Field plugin.

On ACF, you need to call the functions on your templates with the codes, either on normal pages or WooCommerce places.

Now you just need to create the fields an select where you want to show on.

For cat sake, it will save a lot of time and more productive to your working day.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `acf-woocommerce.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create the fields on ACF as normal and select the place you want to show fields on front end.

== Frequently Asked Questions ==

= What are the required plugin? =

To use this plugin, you must install Advaced Custom Field and WooCommerce plugin first. ACF for WooCommerce is the bridge that connect and help you show ACF fields on WooCommerce pages on frontend.

= Does ACF for WooCommerce support conditional logic? =

Yes, it does!

= Can I upload images, file on WooCommerce checkout page? =

Yes, you can!

= Can I add time picker, date pinker on WooCommerce checkout page? =

Yes, you can too!
